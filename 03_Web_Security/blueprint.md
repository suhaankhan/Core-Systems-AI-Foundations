# 🛠️ The Blueprint

Use this as a reference while taking notes. To see how this looks in VS Code, press `Ctrl + Shift + V`.

---

## 1. Document Structure
# Main Title (Use for Video Name)
## Section Heading (Use for Big Topics)
### Subheading (Use for sub-details)

---

## 2. Text Formatting
* **Bold Text**: Use `**text**` for key terms or "Must-Know" concepts.
* *Italic Text*: Use `*text*` for emphasis or side-notes.
* ~~Strikethrough~~: Use `~~text~~` for outdated methods or mistakes.

---

## 3. Lists & Organization
* **Bullet Points**: Use `*` or `-` for quick facts.
* **Numbered Lists**: Use `1.` for step-by-step processes (like the 3-way handshake).
* **Horizontal Rules**: Use `---` to separate different parts of the video.

---

## 4. Researcher Highlights
> **Researcher Note**: Use `> text` for high-level tips or "If-Then" triggers.
> *Example: If I see a 403, I check for header bypasses.*

---

## 5. Technical Snippets (CRITICAL)
**Inline Code**: Use backticks `` `text` `` for status codes like `200 OK` or verbs like `GET`.

**Code Blocks**: Use triple backticks with the language name to format raw HTTP.
```http
GET /api/v1/user HTTP/1.1
Host: target.com
```
