# Core Systems & AI Engineering Foundations

This repository is a technical log of my transition into systems programming. Instead of taking the shortcut through high-level frameworks, I am focusing on C++ to understand how machines actually process logic and manage memory.

## Technical Focus
My work here centers on the fundamentals that define high-performance software:
* Low-level logic: Building precision-heavy mathematical models and converters.
* System Constants: Managing memory-efficient variables and constants like double and const.
* Professional Workflow: Maintaining clean code architecture and standard Git commit practices.

## The Goal
The objective is to master the "guts" of the machine. By building a deep understanding of how data and logic are handled at a systems level, I am preparing to eventually design and scale custom AI/ML architectures from the ground up.

This is an active project with daily pushes. The commit history serves as the live log of my technical progress.
